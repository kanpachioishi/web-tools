一端固定・他端ローラー支持梁の三角形分布荷重（ローラー端最大）

固定端Aとローラー支持Bを持つ1次不静定梁に、固定端A（$x = 0$）で0、ローラー端B（$x = l$）で最大 $q$ となる三角形分布荷重 $w(x) = qx/l$ が作用する場合の反力・せん断力・曲げモーメントを重ね合わせ法で求めます。

問題

長さ $l$ の梁において、左端Aを固定端、右端Bをローラー支持とします。固定端A（$x = 0$）で荷重強度が0、ローラー端B（$x = l$）で最大 $q$ となる三角形分布荷重 $w(x) = qx/l$ が下向きに作用しています。梁の曲げ剛性は $EI$（一様）とします。

この梁は反力が3つ（$R_{\mathrm{A}}$, $J_{\mathrm{A}}$, $R_{\mathrm{B}}$）あるのに対し、釣り合い条件は2つしかないため、1次不静定構造です。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. 反力 $R_{\mathrm{A}},\, R_{\mathrm{B}}$
  2. 固定端モーメント $J_{\mathrm{A}}$

解答

解法の方針（重ね合わせ法）

ローラー支持Bの反力 $R_{\mathrm{B}}$ を冗長力とし、Bの拘束を解除して片持ち梁（固定端A、自由端B）に分解する。

  • 系(1): 外力のみ

    片持ち梁に三角形分布荷重 $w(x) = qx/l$（固定端で0、自由端で最大 $q$）が作用。自由端Bでのたわみ（下向き）:

    $$\delta_q = \frac{11ql^4}{120EI}$$

    （導出は片持ち梁の三角形分布荷重（自由端最大）を参照）

  • 系(2): 冗長力のみ

    片持ち梁のBに上向き力 $R_{\mathrm{B}}$ が作用。自由端Bでのたわみ（上向き）:

    $$\delta_R = \frac{R_{\mathrm{B}} l^3}{3EI}$$
  • 適合条件

    元の構造ではBのたわみがゼロなので:

    $$\delta_q = \delta_R \quad \Rightarrow \quad \frac{11ql^4}{120EI} = \frac{R_{\mathrm{B}} l^3}{3EI}$$

反力の算出

適合条件から冗長力を求める。

$$R_{\mathrm{B}} = \frac{11ql^4}{120EI} \times \frac{3EI}{l^3} = \frac{11ql}{40}$$

三角形分布荷重の合力は $W = ql/2$ であるから、鉛直方向の力の釣り合いより:

$$R_{\mathrm{A}} = \frac{ql}{2} - R_{\mathrm{B}} = \frac{ql}{2} - \frac{11ql}{40} = \frac{20ql - 11ql}{40} = \frac{9ql}{40}$$

A点まわりのモーメントの釣り合いから反偶力（反時計回りを正）を求める。三角形分布荷重 $w(x) = qx/l$ の合力の作用点はAから $2l/3$ の位置にあるので:

$$J_{\mathrm{A}} = \frac{ql}{2} \cdot \frac{2l}{3} - R_{\mathrm{B}} \cdot l = \frac{ql^2}{3} - \frac{11ql^2}{40} = \frac{40ql^2 - 33ql^2}{120} = \frac{7ql^2}{120}$$

反力のまとめ

$$R_{\mathrm{A}} = \frac{9ql}{40}, \quad R_{\mathrm{B}} = \frac{11ql}{40}, \quad J_{\mathrm{A}} = \frac{7ql^2}{120}$$

解答まとめ

一端固定・他端ローラー支持梁・三角形分布荷重（ローラー端最大）の公式

$$R_{\mathrm{A}} = \frac{9ql}{40}, \quad R_{\mathrm{B}} = \frac{11ql}{40}, \quad J_{\mathrm{A}} = \frac{7ql^2}{120}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

固定端Aから右方向に $x$ 軸を取り、任意断面でのせん断力を求める。

$$V(x) = R_{\mathrm{A}} - \int_0^x \frac{qt}{l}\,dt = \frac{9ql}{40} - \frac{qx^2}{2l}$$
  • $x = 0$（固定端A）: $V = \dfrac{9ql}{40}$
  • $x = l$（ローラーB）: $V = \dfrac{9ql}{40} - \dfrac{ql}{2} = -\dfrac{11ql}{40} = -R_{\mathrm{B}}$ ✓
  • ゼロ点: $V = 0$ となる位置は $x = \dfrac{3l}{2\sqrt{5}} \approx 0.671\,l$

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントを求める（下側引張を正とする）。

$$M(x) = -J_{\mathrm{A}} + R_{\mathrm{A}} x - \frac{qx^3}{6l} = -\frac{7ql^2}{120} + \frac{9qlx}{40} - \frac{qx^3}{6l}$$
  • $x = 0$（固定端A）: $M = -\dfrac{7ql^2}{120}$（ホギング）
  • $x = l$（ローラーB）: $M = -\dfrac{7ql^2}{120} + \dfrac{9ql^2}{40} - \dfrac{ql^2}{6} = \dfrac{-7 + 27 - 20}{120}\,ql^2 = 0$ ✓
  • 最大正モーメントの位置: $\dfrac{dM}{dx} = V(x) = 0$ より $x = \dfrac{3l}{2\sqrt{5}} \approx 0.671\,l$
  • 最大正モーメント: $M_{\max} = \dfrac{9ql^2}{40\sqrt{5}} - \dfrac{7ql^2}{120} - \dfrac{9ql^2}{80\sqrt{5}} = \dfrac{9ql^2}{80\sqrt{5}} - \dfrac{7ql^2}{120} \approx 0.042\,ql^2$

ポイント

  • この問題は三角形分布荷重（固定端最大）の荷重の向きを反転させたパターン。荷重がローラー端で最大になるため、$R_{\mathrm{B}}$ が大きくなり反力の配分が変わる。
  • 固定端最大（$w(x) = q(1-x/l)$）では $R_{\mathrm{B}} = ql/10$, $J_{\mathrm{A}} = ql^2/15$ であったのに対し、ローラー端最大（$w(x) = qx/l$）では $R_{\mathrm{B}} = 11ql/40$, $J_{\mathrm{A}} = 7ql^2/120$ と異なる。荷重の重心が右に移動した分、$R_{\mathrm{B}}$ が大きくなっている。
  • 重ね合わせ法では、ローラー反力 $R_{\mathrm{B}}$ を冗長力として取り除き、片持ち梁に分解して考える。片持ち梁のたわみ公式がカギとなる。
  • 片持ち梁の三角形分布荷重（$w(x) = qx/l$、自由端最大）による自由端たわみは $\delta = \dfrac{11ql^4}{120EI}$。$11/120$ という係数は積分計算から導かれる。
  • SFDは2次曲線、BMDは3次曲線になる。SFDのゼロ点（$x = 3l/(2\sqrt{5}) \approx 0.671\,l$）がBMDの極値点に対応する関係は、$V = dM/dx$ から導かれる。
