片持ち梁＋ばね支持の等分布荷重

固定端Aの片持ち梁で、自由端Bをばね定数 $k$ のばねで支持し、全長にわたって等分布荷重 $q$ が作用する不静定問題を、重ね合わせ法（適合条件法）で解きます。

問題

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bをばね定数 $k$ のばねで支持します。梁の全長にわたって等分布荷重 $q$（単位長さあたりの荷重、下向き）が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が、ばね支持端Bにはばね反力 $R_{\mathrm{B}}$ が発生します。未知反力が3つに対して釣り合い式が2つなので、1次不静定です。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. ばね反力 $R_{\mathrm{B}}$
  2. 固定端反力 $R_{\mathrm{A}}$ および固定端モーメント $J_{\mathrm{A}}$
  3. 自由端Bのたわみ $y_B$

解答

解法の方針 — 重ね合わせ法（適合条件法）

この問題は1次不静定であり、力の釣り合いだけでは反力を決定できません。そこで重ね合わせ法（適合条件法）を用います。

考え方は次の通りです:

  • 余剰拘束を取り除く
    ばね支持を取り除き、B端を自由端とした静定な片持ち梁を基本系とする。ばね反力 $R_{\mathrm{B}}$ を余剰未知力として扱う。
  • 外力による自由端のたわみ（ケース1）
    ばねのない片持ち梁に等分布荷重 $q$ が全長に作用したとき、自由端Bでのたわみ $\delta_q$ を求める。
  • 余剰力によるたわみ（ケース2）
    ばねのない片持ち梁にB端で上向きの力 $R_{\mathrm{B}}$ が作用したとき、Bでの上向きたわみ $\delta_R$ を求める。
  • 適合条件（変位の整合性）
    B端の実際のたわみ（下向き）がばねの伸びに等しいという条件を立て、$R_{\mathrm{B}}$ を決定する。 $$\delta_q - \delta_R = \frac{R_{\mathrm{B}}}{k}$$

反力の算出

ケース1: 等分布荷重による自由端Bのたわみ

片持ち梁に等分布荷重 $q$ が全長に作用するとき、自由端（$x = l$）のたわみは:

$$\delta_q = \frac{ql^4}{8EI}$$

ケース2: ばね反力 $R_{\mathrm{B}}$ による自由端Bの上向きたわみ

自由端Bに上向きの力 $R_{\mathrm{B}}$ が作用するとき、Bでのたわみは:

$$\delta_R = \frac{R_{\mathrm{B}} l^3}{3EI}$$

適合条件

B端の実際のたわみ（下向き）はばねの伸びに等しいから:

$$\delta_q - \delta_R = \frac{R_{\mathrm{B}}}{k}$$

代入して整理すると:

$$\frac{ql^4}{8EI} - \frac{R_{\mathrm{B}} l^3}{3EI} = \frac{R_{\mathrm{B}}}{k}$$ $$\frac{ql^4}{8EI} = R_{\mathrm{B}} \left(\frac{l^3}{3EI} + \frac{1}{k}\right) = R_{\mathrm{B}} \cdot \frac{kl^3 + 3EI}{3kEI}$$ $$R_{\mathrm{B}} = \frac{ql^4}{8EI} \cdot \frac{3kEI}{kl^3 + 3EI} = \frac{3kql^4}{8(kl^3 + 3EI)}$$

ばね反力

$$R_{\mathrm{B}} = \frac{3kql^4}{8(3EI + kl^3)}$$

固定端の反力

鉛直方向の釣り合いとA点まわりのモーメントの釣り合いから:

$$R_{\mathrm{A}} = ql - R_{\mathrm{B}}$$ $$J_{\mathrm{A}} = \frac{ql^2}{2} - R_{\mathrm{B}} l$$

たわみ

たわみは重ね合わせの原理により、等分布荷重 $q$ によるたわみからばね反力 $R_{\mathrm{B}}$ によるたわみを差し引くことで求められます。

自由端Bのたわみ

B端のたわみはばねの伸びに等しく:

$$y_B = \frac{R_{\mathrm{B}}}{k} = \frac{3ql^4}{8(3EI + kl^3)}$$

たわみの一般式

等分布荷重による片持ち梁のたわみ:

$$y_q(x) = \frac{q}{24EI}(x^4 - 4lx^3 + 6l^2 x^2)$$

自由端の集中荷重 $R_{\mathrm{B}}$ による片持ち梁のたわみ:

$$y_R(x) = \frac{R_{\mathrm{B}} x^2}{6EI}(3l - x)$$

実際のたわみ:

$$y(x) = y_q(x) - y_R(x) = \frac{q}{24EI}(x^4 - 4lx^3 + 6l^2 x^2) - \frac{R_{\mathrm{B}} x^2}{6EI}(3l - x)$$

解答まとめ

片持ち梁＋ばね支持・等分布荷重の公式

$$R_{\mathrm{B}} = \frac{3kql^4}{8(3EI + kl^3)}$$ $$R_{\mathrm{A}} = ql - R_{\mathrm{B}}$$ $$J_{\mathrm{A}} = \frac{ql^2}{2} - R_{\mathrm{B}} l$$ $$y_B = \frac{R_{\mathrm{B}}}{k}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

固定端Aから右向きに $x$ を取り、左側の釣り合いを考えます。

$$V(x) = R_{\mathrm{A}} - qx$$
  • $x = 0$: $V = R_{\mathrm{A}} = ql - R_{\mathrm{B}}$
  • $x = l$: $V = R_{\mathrm{A}} - ql = -R_{\mathrm{B}}$（ばね反力で打ち消される）

せん断力は $R_{\mathrm{A}}$ から線形に減少し、$V = 0$ となる位置 $x_0 = R_{\mathrm{A}} / q$ で曲げモーメントが極値をとります。

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメント:

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}} x + \frac{qx^2}{2}$$

すなわち:

$$M(x) = \left(\frac{ql^2}{2} - R_{\mathrm{B}} l\right) - (ql - R_{\mathrm{B}})x + \frac{qx^2}{2}$$
  • $x = 0$: $M = J_{\mathrm{A}} = ql^2/2 - R_{\mathrm{B}} l$（反偶力）
  • $x = l$: $M = J_{\mathrm{A}} - R_{\mathrm{A}} l + ql^2/2 = 0$（ばね支持端はモーメント自由）
  • $V = 0$ となる $x_0 = R_{\mathrm{A}} / q$ で曲げモーメントが極値

ポイント

  • 等分布荷重の場合も、重ね合わせ法の手順は集中荷重の場合と同じ。「ばねを外す → たわみを2ケース計算 → 適合条件で $R_{\mathrm{B}}$ を決定」という流れを覚えておこう。
  • $k \to \infty$ のとき: $R_{\mathrm{B}} \to \frac{3ql}{8}$ となり、一端固定・他端支持梁（propped cantilever）の等分布荷重の解に一致する。
  • $k \to 0$ のとき: $R_{\mathrm{B}} \to 0$ となり、純粋な片持ち梁の等分布荷重の解に一致する。
  • SFDが線形でBMDが放物線となるのは等分布荷重の特徴。集中荷重の場合（SFDが階段、BMDが折れ線）と比較して整理するとよい。
  • せん断力が0となる位置 $x_0 = R_{\mathrm{A}} / q$ で曲げモーメントが極大値をとる。この関係は等分布荷重の問題では常に成り立つ。
  • ばね定数 $k$ と梁の剛性 $EI/l^3$ の比が反力配分を決める。次元解析で $kl^3/(EI)$ が無次元パラメータとなることを確認しよう。
