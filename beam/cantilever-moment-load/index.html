片持ち梁のモーメント荷重（自由端）

固定端Aと自由端Bを持つ片持ち梁の自由端Bにモーメント $M_0$（時計回り）が作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

目次

  1. 問題
  2. 解答
  3. 補足：せん断力図・曲げモーメント図
  4. ポイント

問題

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bを自由端とします。自由端Bに時計回りのモーメント $M_0$ が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. 反力 $R_{\mathrm{A}}$ と固定端モーメント（反偶力）$J_{\mathrm{A}}$
  2. 自由端のたわみ $\delta_{\max}$

解答

反力の算出

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が発生します。力の釣り合いとモーメントの釣り合いから求めます。

  • 鉛直方向の力の釣り合い

    外力に鉛直方向の力がないため:

    $$R_{\mathrm{A}} = 0$$
  • A点まわりのモーメントの釣り合い

    反時計回りを正とします。自由端Bの時計回りモーメント $M_0$ に対して:

    $$J_{\mathrm{A}} - M_0 = 0 \quad \Rightarrow \quad J_{\mathrm{A}} = M_0$$

モーメント荷重のみが作用する場合、鉛直反力はゼロとなります。固定端には外力のモーメントと等しい反モーメントだけが発生します。

たわみとたわみ角

たわみの微分方程式 $EI\,y'' = M(x)$ を解きます。境界条件は固定端（$x = 0$）でたわみ $y = 0$、たわみ角 $y' = 0$ です。

微分方程式の解

$$EI\,y'' = M_0$$

1回積分してたわみ角:

$$EI\,y' = M_0 x + C_1$$

境界条件 $y'(0) = 0$ より $C_1 = 0$。

$$EI\,y' = M_0 x$$

もう1回積分してたわみ:

$$EI\,y = \frac{M_0 x^2}{2} + C_2$$

境界条件 $y(0) = 0$ より $C_2 = 0$。

結果

たわみ:

$$y(x) = \frac{M_0 x^2}{2EI}$$

たわみ角:

$$\theta(x) = y'(x) = \frac{M_0 x}{EI}$$

最大たわみ（自由端 $x = l$）:

$$\delta_{\max} = y(l) = \frac{M_0 l^2}{2EI}$$

最大たわみ角（自由端 $x = l$）:

$$\theta_{\max} = \theta(l) = \frac{M_0 l}{EI}$$

解答まとめ

片持ち梁・自由端モーメント荷重の解答

$$R_{\mathrm{A}} = 0, \qquad J_{\mathrm{A}} = M_0$$ $$\delta_{\max} = \frac{M_0 l^2}{2EI}, \qquad \theta_{\max} = \frac{M_0 l}{EI}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

鉛直方向の外力が存在しないため、梁のどの位置で切断してもせん断力はゼロです。

$$V(x) = 0 \quad (0 \le x \le l)$$

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントを考えます。せん断力がゼロであるため、曲げモーメントは変化しません。

$$M(x) = M_0 \quad (0 \le x \le l)$$

曲げモーメントは梁全長にわたって一定値 $M_0$ です。

ポイント

  • モーメント荷重のみの場合、せん断力がゼロになるのが大きな特徴です。$\frac{dM}{dx} = V = 0$ という関係から、曲げモーメントが一定になることが分かります。
  • 曲げモーメントが一定なので、梁は純粋曲げ（pure bending）の状態にあります。これは梁が円弧状にたわむことを意味します。
  • たわみは $x$ の2次関数（放物線）です。モーメント一定のもとでの積分が非常にシンプルになるため、微分方程式の練習問題として最適です。
  • $\delta_{\max} = \frac{M_0 l^2}{2EI}$ は、集中荷重の場合の $\frac{Pl^3}{3EI}$ と比較して覚えるとよいでしょう。
  • この問題パターンは、重ね合わせの原理で他の荷重条件と組み合わせて使うことが多いです。
