片持ち梁の等分布荷重（右半分・自由端側）

固定端Aと自由端Bを持つ片持ち梁の右半分（自由端側、$l/2 \le x \le l$）にのみ等分布荷重 $q$ が作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

問題

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bを自由端とします。右半分（$l/2 \le x \le l$、自由端側）に下向きの等分布荷重 $q$ が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. 反力 $R_{\mathrm{A}}$ と固定端モーメント（反偶力）$J_{\mathrm{A}}$
  2. 自由端のたわみ $\delta_{\max}$

解答

等分布荷重 $q$ が右半分に作用するので、合力は $ql/2$ となり、その作用点は $x = 3l/4$（右半分区間の中央）です。

反力の算出

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が発生します。

  • 鉛直方向の力の釣り合い $$R_{\mathrm{A}} - \frac{ql}{2} = 0 \quad \Rightarrow \quad \boxed{R_{\mathrm{A}} = \frac{ql}{2}}$$
  • A点まわりのモーメントの釣り合い

    反時計回りを正とします。合力 $ql/2$ は $x = 3l/4$ に作用します。

    $$J_{\mathrm{A}} - \frac{ql}{2} \cdot \frac{3l}{4} = 0 \quad \Rightarrow \quad \boxed{J_{\mathrm{A}} = \frac{3ql^2}{8}}$$

たわみとたわみ角

たわみの微分方程式 $-EI\,y'' = M(x)$ を各区間で解きます。境界条件は固定端（$x = 0$）でたわみ $y = 0$、たわみ角 $y' = 0$ です。また、$x = l/2$ での連続条件（たわみと傾きが一致）を用います。

曲げモーメント

位置 $x$ で梁を切断し、左側の釣り合いから:

区間1（$0 \le x \le l/2$）: 荷重領域の手前なので:

$$M(x) = R_{\mathrm{A}}\,x - J_{\mathrm{A}} = \frac{ql}{2}x - \frac{3ql^2}{8}$$

区間2（$l/2 \le x \le l$）:

$$M(x) = R_{\mathrm{A}}\,x - J_{\mathrm{A}} - \frac{q(x - l/2)^2}{2} = -\frac{q(l-x)^2}{2}$$

区間1（$0 \le x \le l/2$）の解

$$-EI\,y'' = \frac{ql}{2}\,x - \frac{3ql^2}{8}$$

1回積分（境界条件 $y'(0)=0$ より積分定数 $=0$）:

$$-EI\,y' = -\frac{3ql^2}{8}\,x + \frac{ql}{4}\,x^2$$

もう1回積分（境界条件 $y(0)=0$ より積分定数 $=0$）:

$$-EI\,y = -\frac{3ql^2}{16}\,x^2 + \frac{ql}{12}\,x^3$$

$x = l/2$ での値（区間2の境界条件として使用）:

$$-EI\,y\!\left(\frac{l}{2}\right) = -\frac{3ql^2}{16}\cdot\frac{l^2}{4} + \frac{ql}{12}\cdot\frac{l^3}{8} = -\frac{3ql^4}{64} + \frac{ql^4}{96} = -\frac{7ql^4}{192}$$ $$-EI\,y'\!\left(\frac{l}{2}\right) = -\frac{3ql^2}{8}\cdot\frac{l}{2} + \frac{ql}{4}\cdot\frac{l^2}{4} = -\frac{3ql^3}{16} + \frac{ql^3}{16} = -\frac{ql^3}{8}$$

区間2（$l/2 \le x \le l$）の解

$$-EI\,y'' = -\frac{q(l-x)^2}{2}$$

1回積分（積分定数を $D_1$）:

$$-EI\,y' = \frac{q(l-x)^3}{6} + D_1$$

連続条件 $-EI\,y'(l/2) = -ql^3/8$ を適用:

$$-\frac{ql^3}{8} = \frac{q(l/2)^3}{6} + D_1 = \frac{ql^3}{48} + D_1 \quad \Rightarrow \quad D_1 = -\frac{ql^3}{8} - \frac{ql^3}{48} = -\frac{7ql^3}{48}$$ $$-EI\,y' = \frac{q(l-x)^3}{6} - \frac{7ql^3}{48}$$

もう1回積分（積分定数を $D_2$）:

$$-EI\,y = -\frac{q(l-x)^4}{24} - \frac{7ql^3 x}{48} + D_2$$

連続条件 $-EI\,y(l/2) = -7ql^4/192$ を適用:

$$-\frac{7ql^4}{192} = -\frac{q(l/2)^4}{24} - \frac{7ql^3}{48}\cdot\frac{l}{2} + D_2 = -\frac{ql^4}{384} - \frac{7ql^4}{96} + D_2$$ $$D_2 = -\frac{7ql^4}{192} + \frac{ql^4}{384} + \frac{7ql^4}{96} = \frac{-14ql^4 + ql^4 + 28ql^4}{384} = \frac{15ql^4}{384} = \frac{5ql^4}{128}$$ $$-EI\,y = -\frac{q(l-x)^4}{24} - \frac{7ql^3 x}{48} + \frac{5ql^4}{128}$$

最大たわみ・最大たわみ角（自由端 $x = l$）

最大たわみ:

$x = l$ を代入すると $(l-x)^4 = 0$ なので:

$$-EI\,y(l) = 0 - \frac{7ql^4}{48} + \frac{5ql^4}{128} = -\frac{7 \times 128 - 5 \times 48}{48 \times 128}\,ql^4 = -\frac{896 - 240}{6144}\,ql^4 = -\frac{41ql^4}{384}$$ $$\boxed{\delta_{\max} = y(l) = \frac{41ql^4}{384EI}}$$

最大たわみ角:

$x = l$ を代入すると $(l-x)^3 = 0$ なので:

$$-EI\,y'(l) = 0 - \frac{7ql^3}{48} \quad \Rightarrow \quad \boxed{\theta_{\max} = y'(l) = \frac{7ql^3}{48EI}}$$

解答まとめ

片持ち梁・右半分等分布荷重（自由端側）の解答

$$R_{\mathrm{A}} = \frac{ql}{2}, \qquad J_{\mathrm{A}} = \frac{3ql^2}{8}$$ $$\delta_{\max} = \frac{41ql^4}{384EI}, \qquad \theta_{\max} = \frac{7ql^3}{48EI}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

任意の位置 $x$ で梁を切断し、左側の釣り合いを考えます。区間によって式が変わります。

区間1（$0 \le x < l/2$）: 荷重なし

$$V(x) = R_{\mathrm{A}} = \frac{ql}{2} \quad \text{（一定）}$$

区間2（$l/2 \le x \le l$）: 分布荷重が $x = l/2$ から $x$ まで作用

$$V(x) = R_{\mathrm{A}} - q\!\left(x - \frac{l}{2}\right) = \frac{ql}{2} - q\!\left(x - \frac{l}{2}\right) = q(l - x)$$

自由端Bでは $V(l) = 0$（自由端条件を満たす）。

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントも区間ごとに異なります。

区間1（$0 \le x \le l/2$）:

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}}\,x = \frac{3ql^2}{8} - \frac{ql}{2}\,x \quad \text{（線形）}$$

区間2（$l/2 \le x \le l$）: 左側の分布荷重部分も含めて考えます。

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}}\,x + q\!\left(x - \frac{l}{2}\right)\!\cdot\!\frac{x - l/2}{2} = \frac{q(l-x)^2}{2} \quad \text{（2次曲線）}$$

主要点の確認:

  • $x = 0$（固定端A）: $M = 3ql^2/8$（最大）
  • $x = l/2$（中央）: 区間1より $M = 3ql^2/8 - ql^2/4 = ql^2/8$、区間2より $M = q(l/2)^2/2 = ql^2/8$（一致）
  • $x = l$（自由端B）: $M = 0$

ポイント

  • 荷重位置が自由端側ほどたわみが大きい。 右半分（自由端側）に荷重が作用する本問では $\delta_{\max} = 41ql^4/384EI$ となる。一方、左半分（固定端側）に同じ強度の荷重を与えた場合は $7ql^4/384EI$ であり、自由端側への荷重は約6倍近くも大きなたわみを生む。
  • また全長等分布荷重 $q$ のたわみ $ql^4/(8EI) = 48ql^4/384EI$ と比較すると、右半分だけで $41/48 \approx 85\%$ のたわみが発生することになり、自由端側の荷重がたわみに与える寄与の大きさが分かる。
  • 区間が分かれる問題では、$x = l/2$ での連続条件（たわみの値・傾きが一致）を使って積分定数を決定するのがポイント。
  • 区間2のBMD $M = q(l-x)^2/2$ は自由端から測った距離 $(l-x)$ で整理すると非常にシンプルな形になる。自由端側から解析するアプローチも有効。
  • せん断力が区間1で一定（$ql/2$）、区間2で線形減少（$q(l-x)$）になるのは、荷重が存在しない区間では $V$ が一定、分布荷重がある区間では $dV/dx = -q$ の関係による。
