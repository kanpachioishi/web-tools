片持ち梁の等分布荷重（左半分・固定端側）

固定端Aと自由端Bを持つ片持ち梁において、固定端側の左半分（$0 \le x \le l/2$）にのみ等分布荷重 $q$ が作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

問題

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bを自由端とします。固定端側の左半分（$0 \le x \le l/2$）に強度 $q$（単位長さあたり）の等分布荷重が下向きに作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. 反力 $R_{\mathrm{A}}$ と固定端モーメント（反偶力）$J_{\mathrm{A}}$
  2. 自由端のたわみ $\delta_{\max}$

解答

等分布荷重の合力は $ql/2$（下向き）で、その作用点はA端から $l/4$ の位置です。

反力の算出

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が発生します。

  • 鉛直方向の力の釣り合い $$R_{\mathrm{A}} - \frac{ql}{2} = 0 \quad \Rightarrow \quad R_{\mathrm{A}} = \frac{ql}{2}$$
  • A点まわりのモーメントの釣り合い

    反時計回りを正とします。合力 $ql/2$ はA端から $l/4$ の位置に作用します。

    $$J_{\mathrm{A}} - \frac{ql}{2} \cdot \frac{l}{4} = 0 \quad \Rightarrow \quad J_{\mathrm{A}} = \frac{ql^2}{8}$$

たわみとたわみ角

たわみの微分方程式 $-EI\,y'' = M(x)$ を区間ごとに解きます。境界条件は固定端（$x = 0$）でたわみ $y = 0$、たわみ角 $y' = 0$、また $x = l/2$ での連続条件（たわみとたわみ角が一致）を用います。

曲げモーメント

位置 $x$ で梁を切断し、左側の釣り合いから:

区間1（$0 \le x \le l/2$）:

$$M(x) = R_{\mathrm{A}}\,x - J_{\mathrm{A}} - \frac{qx^2}{2} = -\frac{ql^2}{8} + \frac{ql}{2}x - \frac{qx^2}{2}$$

区間2（$l/2 < x \le l$）: 荷重領域を過ぎているため $M(x) = 0$。

区間1：$0 \le x \le l/2$

$$-EI\,y'' = -\frac{ql^2}{8} + \frac{qlx}{2} - \frac{qx^2}{2}$$

1回積分してたわみ角:

$$-EI\,y' = -\frac{ql^2}{8}x + \frac{qlx^2}{4} - \frac{qx^3}{6} + C_1$$

境界条件 $y'(0) = 0$ より $C_1 = 0$。

もう1回積分してたわみ:

$$-EI\,y = -\frac{ql^2 x^2}{16} + \frac{qlx^3}{12} - \frac{qx^4}{24} + C_2$$

境界条件 $y(0) = 0$ より $C_2 = 0$。

よって区間1のたわみとたわみ角:

$$y(x) = \frac{q}{EI}\left(\frac{l^2 x^2}{16} - \frac{lx^3}{12} + \frac{x^4}{24}\right) \quad (0 \le x \le l/2)$$ $$y'(x) = \frac{q}{EI}\left(\frac{l^2 x}{8} - \frac{lx^2}{4} + \frac{x^3}{6}\right)$$

区間2：$l/2 < x \le l$

この区間では $M(x) = 0$ なので:

$$-EI\,y'' = 0 \quad \Rightarrow \quad y' = A_1,\quad y = A_1 x + A_2 \quad \text{（線形）}$$

連続条件（$x = l/2$ での接続）

まず $x = l/2$ でのたわみ角を区間1の式から求めます:

$$y'\!\left(\frac{l}{2}\right) = \frac{q}{EI}\left(\frac{l^2}{8}\cdot\frac{l}{2} - \frac{l}{4}\cdot\frac{l^2}{4} + \frac{l^3}{48}\right) = \frac{q}{EI}\cdot\frac{l^3}{48} = \frac{ql^3}{48EI}$$

次に $x = l/2$ でのたわみを区間1から求めます:

$$y\!\left(\frac{l}{2}\right) = \frac{q}{EI}\left(\frac{l^2}{16}\cdot\frac{l^2}{4} - \frac{l}{12}\cdot\frac{l^3}{8} + \frac{l^4}{384}\right) = \frac{q}{EI}\cdot\frac{l^4}{128} = \frac{ql^4}{128EI}$$

区間2は直線（たわみ角一定）なので、$A_1 = ql^3/(48EI)$。連続条件 $y(l/2) = ql^4/(128EI)$ から $A_2$ を求めます:

$$\frac{ql^3}{48EI}\cdot\frac{l}{2} + A_2 = \frac{ql^4}{128EI} \quad \Rightarrow \quad A_2 = \frac{ql^4}{128EI} - \frac{ql^4}{96EI} = -\frac{ql^4}{384EI}$$

よって区間2のたわみ:

$$y(x) = \frac{q}{EI}\left(\frac{l^3 x}{48} - \frac{l^4}{384}\right) \quad (l/2 < x \le l)$$

最大たわみと最大たわみ角

自由端 $x = l$ でのたわみ（最大たわみ）:

$$\delta_{\max} = y(l) = \frac{q}{EI}\left(\frac{l^4}{48} - \frac{l^4}{384}\right) = \frac{q}{EI}\cdot\frac{7l^4}{384} = \frac{7ql^4}{384EI}$$

右半分ではたわみ角が一定（剛体回転）なので、最大たわみ角:

$$\theta_{\max} = y'(l) = \frac{ql^3}{48EI}$$

解答まとめ

片持ち梁・固定端側等分布荷重（左半分）の解答

$$R_{\mathrm{A}} = \frac{ql}{2}, \qquad J_{\mathrm{A}} = \frac{ql^2}{8}$$ $$\delta_{\max} = \frac{7ql^4}{384EI}, \qquad \theta_{\max} = \frac{ql^3}{48EI}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

任意の位置 $x$ で梁を切断し、左側の釣り合いを考えます。

左半分（$0 \le x \le l/2$）: 区間に分布荷重 $q$ が存在するため:

$$V(x) = R_{\mathrm{A}} - qx = \frac{ql}{2} - qx$$
  • $x = 0$（固定端A）: $V = ql/2$
  • $x = l/2$（中点）: $V = ql/2 - q \cdot l/2 = 0$

右半分（$l/2 < x \le l$）: 荷重がないため:

$$V(x) = 0$$

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントは:

左半分（$0 \le x \le l/2$）:

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}} x + \frac{qx^2}{2} = \frac{ql^2}{8} - \frac{qlx}{2} + \frac{qx^2}{2}$$
  • $x = 0$: $M = ql^2/8$（最大）
  • $x = l/2$: $M = ql^2/8 - ql^2/4 + ql^2/8 = 0$

右半分（$l/2 < x \le l$）: 荷重がなくせん断力も 0 なので:

$$M(x) = 0$$

曲げモーメントは固定端で最大 $ql^2/8$ となり、中点に向かって放物線状に減少し、中点以降は 0 です。

ポイント

  • 荷重が固定端側（左半分）に集中しているため、自由端側（右半分）に荷重が作用する場合と比べて最大たわみが小さくなります。全長等分布荷重の場合 $\delta = ql^4/(8EI)$、右半分荷重の場合と比較すると、固定端に近いほど梁が拘束されており変形しにくいことがわかります。
  • 右半分（$l/2 < x \le l$）では荷重もせん断力もゼロなため、たわみ曲線は直線（剛体的な回転）になります。これは連続条件を用いて区間2の積分定数を決めるポイントです。
  • 固定端でのモーメント $J_{\mathrm{A}} = ql^2/8$ は、全長等分布荷重の場合の $ql^2/2$ と比べて $1/4$ に過ぎません。荷重範囲が短く、かつ固定端に近いほどモーメントアームが小さいためです。
  • たわみの微分方程式を区間ごとに解く際は、接続点でのたわみと傾きの連続条件を必ず確認することが重要です。
