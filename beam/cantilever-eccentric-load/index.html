片持ち梁の偏心集中荷重（任意点）

固定端Aと自由端Bを持つ片持ち梁の固定端から距離 $a$ の任意点Cに集中荷重 $P$ が作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

  4. ポイント

問題

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bを自由端とします。固定端Aから距離 $a$（$0 < a \le l$）の点Cに下向きの集中荷重 $P$ が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. 反力 $R_{\mathrm{A}}$ と固定端モーメント（反偶力）$J_{\mathrm{A}}$
  2. 自由端のたわみ $\delta_{\max}$

解答

反力の算出

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が発生します。力の釣り合いとモーメントの釣り合いから求めます。

  • 鉛直方向の力の釣り合い $$R_{\mathrm{A}} - P = 0 \quad \Rightarrow \quad R_{\mathrm{A}} = P$$
  • A点まわりのモーメントの釣り合い

    反時計回りを正とします。荷重 $P$ はA点から距離 $a$ の位置に作用します。

    $$J_{\mathrm{A}} - P \cdot a = 0 \quad \Rightarrow \quad J_{\mathrm{A}} = Pa$$

自由端荷重（$a = l$）の場合は $J_{\mathrm{A}} = Pl$ となり、既知の公式に一致します。

たわみとたわみ角

荷重点Cで梁を2区間に分けて解きます。境界条件は固定端（$x = 0$）でたわみ $y = 0$、たわみ角 $y' = 0$、荷重点C（$x = a$）での連続条件（たわみとたわみ角が一致）です。

区間 1：$0 \le x \le a$

この区間の曲げモーメントは $M(x) = P(a - x)$ なので、たわみの微分方程式は:

$$EI\,y_1'' = P(a - x)$$

1回積分してたわみ角:

$$EI\,y_1' = P\left(ax - \frac{x^2}{2}\right) + C_1$$

境界条件 $y_1'(0) = 0$ より $C_1 = 0$。

$$EI\,y_1' = P\left(ax - \frac{x^2}{2}\right)$$

もう1回積分してたわみ:

$$EI\,y_1 = P\left(\frac{ax^2}{2} - \frac{x^3}{6}\right) + C_2$$

境界条件 $y_1(0) = 0$ より $C_2 = 0$。

$$\boxed{y_1(x) = \frac{P}{EI}\left(\frac{ax^2}{2} - \frac{x^3}{6}\right)} \quad (0 \le x \le a)$$

区間 2：$a < x \le l$

この区間の曲げモーメントは $M(x) = 0$ なので:

$$EI\,y_2'' = 0$$

積分すると $y_2'' = 0$ より $y_2$ は一次関数です。

$$EI\,y_2' = C_3, \qquad EI\,y_2 = C_3 x + C_4$$

連続条件から $C_3$, $C_4$ を求めます。$x = a$ でのたわみ角：

$$C_3 = EI\,y_1'(a) = P\left(a \cdot a - \frac{a^2}{2}\right) = \frac{Pa^2}{2}$$

$x = a$ でのたわみ：

$$EI\,y_1(a) = P\left(\frac{a^3}{2} - \frac{a^3}{6}\right) = \frac{Pa^3}{3}$$ $$C_4 = EI\,y_1(a) - C_3 \cdot a = \frac{Pa^3}{3} - \frac{Pa^2}{2} \cdot a = -\frac{Pa^3}{6}$$

したがって:

$$EI\,y_2 = \frac{Pa^2}{2} x - \frac{Pa^3}{6}$$ $$\boxed{y_2(x) = \frac{Pa^2(3x - a)}{6EI}} \quad (a < x \le l)$$

主要な値

荷重点Cのたわみ（$x = a$）:

$$y(a) = \frac{Pa^3}{3EI}$$

最大たわみ角（荷重点C以降は一定）:

$$\theta_{\max} = y_1'(a) = \frac{Pa^2}{2EI}$$

最大たわみ（自由端 $x = l$）:

$$\delta_{\max} = y_2(l) = \frac{Pa^2(3l - a)}{6EI}$$

解答まとめ

片持ち梁・任意点集中荷重の解答

$$R_{\mathrm{A}} = P, \qquad J_{\mathrm{A}} = Pa$$ $$\delta_{\max} = \frac{Pa^2(3l - a)}{6EI}, \qquad \theta_{\max} = \frac{Pa^2}{2EI}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

梁を任意の位置 $x$ で切断し、左側の釣り合いを考えます。

荷重点Cより左側 $0 \le x \le a$：

$$V(x) = R_{\mathrm{A}} = P$$

荷重点Cより右側 $a < x \le l$：

$$V(x) = R_{\mathrm{A}} - P = P - P = 0$$

荷重点Cより左は $P$ 一定、右はゼロになります。

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントを求めます。

荷重点Cより左側 $0 \le x \le a$：

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}} \cdot x = Pa - Px = P(a - x)$$

荷重点Cより右側 $a < x \le l$：

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}} \cdot x + P(x - a) = Pa - Px + Px - Pa = 0$$
  • $x = 0$（固定端A）: $M = Pa$（最大）
  • $x = a$（荷重点C）: $M = 0$
  • $x = a$ から $x = l$（自由端B）: $M = 0$（一定）

曲げモーメントは固定端で最大 $Pa$ となり、荷重点Cまで線形に減少し、そこから右はゼロです。

ポイント

  • これは一般解（汎用公式）です。荷重位置 $a$ を変えるだけで様々なケースを網羅できます。
  • $a = l$（自由端荷重）と置くと: $$\delta_{\max} = \frac{Pl^2(3l - l)}{6EI} = \frac{Pl^3}{3EI}, \quad \theta_{\max} = \frac{Pl^2}{2EI}$$ これは片持ち梁の自由端集中荷重の公式に一致します。
  • $a = l/2$（中央荷重）と置くと: $$\delta_{\max} = \frac{P(l/2)^2(3l - l/2)}{6EI} = \frac{5Pl^3}{48EI}$$ こちらも既知の公式に一致します。
  • 荷重点Cより右の区間（$a < x \le l$）は曲げモーメントがゼロのため、梁は直線的に変形します（たわみが $x$ の一次関数になる）。
  • 最大たわみは常に自由端Bで生じ、最大たわみ角は荷重点C以降一定（$= Pa^2/(2EI)$）です。
