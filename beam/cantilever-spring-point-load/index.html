片持ち梁＋ばね支持の集中荷重（中央）

固定端Aの片持ち梁で、自由端Bをばね定数 $k$ のばねで支持し、中央C（$x = l/2$）に集中荷重 $P$ が下向きに作用する不静定問題を、重ね合わせ法（適合条件法）で解きます。

問題

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bをばね定数 $k$ のばねで支持します。梁の中央C（$x = l/2$）に下向きの集中荷重 $P$ が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が、ばね支持端Bにはばね反力 $R_{\mathrm{B}}$ が発生します。未知反力が3つ（$R_{\mathrm{A}}$, $J_{\mathrm{A}}$, $R_{\mathrm{B}}$）に対して釣り合い式が2つなので、1次不静定です。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. ばね反力 $R_{\mathrm{B}}$
  2. 固定端反力 $R_{\mathrm{A}}$ および固定端モーメント $J_{\mathrm{A}}$
  3. 自由端Bのたわみ $y_B$

解答

解法の方針 — 重ね合わせ法（適合条件法）

この問題は1次不静定であり、力の釣り合いだけでは反力を決定できません。そこで重ね合わせ法（適合条件法）を用います。

考え方は次の通りです:

  • 余剰拘束を取り除く
    ばね支持を取り除き、B端を自由端とした静定な片持ち梁を基本系とする。ばね反力 $R_{\mathrm{B}}$ を余剰未知力として扱う。
  • 外力による自由端のたわみ（ケース1）
    ばねのない片持ち梁に荷重 $P$ が $x = l/2$ に作用したとき、自由端Bでのたわみ $\delta_P$ を求める。
  • 余剰力によるたわみ（ケース2）
    ばねのない片持ち梁にB端で上向きの力 $R_{\mathrm{B}}$ が作用したとき、Bでの上向きたわみ $\delta_R$ を求める。
  • 適合条件（変位の整合性）
    B端のたわみ（下向き）とばねの伸び（$R_{\mathrm{B}} / k$）が一致する条件を立て、$R_{\mathrm{B}}$ を決定する。 $$\delta_P - \delta_R = \frac{R_{\mathrm{B}}}{k}$$

反力の算出

ケース1: 荷重Pによる自由端Bのたわみ

片持ち梁の任意位置 $a$（固定端から）に集中荷重 $P$ が作用するとき、自由端（$x = l$）のたわみは:

$$\delta = \frac{Pa^2}{6EI}(3l - a)$$

$a = l/2$ を代入すると:

$$\delta_P = \frac{P(l/2)^2}{6EI}\left(3l - \frac{l}{2}\right) = \frac{Pl^2}{24EI} \cdot \frac{5l}{2} = \frac{5Pl^3}{48EI}$$

ケース2: ばね反力 $R_{\mathrm{B}}$ による自由端Bの上向きたわみ

自由端Bに上向きの力 $R_{\mathrm{B}}$ が作用するとき、Bでのたわみは:

$$\delta_R = \frac{R_{\mathrm{B}} l^3}{3EI}$$

適合条件

B端の実際のたわみ（下向き）はばねの伸びに等しいから:

$$\delta_P - \delta_R = \frac{R_{\mathrm{B}}}{k}$$

代入して整理すると:

$$\frac{5Pl^3}{48EI} - \frac{R_{\mathrm{B}} l^3}{3EI} = \frac{R_{\mathrm{B}}}{k}$$ $$\frac{5Pl^3}{48EI} = R_{\mathrm{B}} \left(\frac{l^3}{3EI} + \frac{1}{k}\right) = R_{\mathrm{B}} \cdot \frac{kl^3 + 3EI}{3kEI}$$ $$R_{\mathrm{B}} = \frac{5Pl^3}{48EI} \cdot \frac{3kEI}{kl^3 + 3EI} = \frac{5Pkl^3}{16(kl^3 + 3EI)}$$

これを整理すると:

ばね反力

$$R_{\mathrm{B}} = \frac{5Pkl^3}{48EI + 16kl^3}$$

固定端の反力

鉛直方向の釣り合いとA点まわりのモーメントの釣り合いから:

$$R_{\mathrm{A}} = P - R_{\mathrm{B}}$$ $$J_{\mathrm{A}} = P \cdot \frac{l}{2} - R_{\mathrm{B}} \cdot l = \frac{Pl}{2} - R_{\mathrm{B}} l$$

たわみ

たわみは重ね合わせの原理により、荷重 $P$ によるたわみからばね反力 $R_{\mathrm{B}}$ によるたわみを差し引くことで求められます。

自由端Bのたわみ

B端のたわみはばねの伸びに等しく:

$$y_B = \frac{R_{\mathrm{B}}}{k} = \frac{5Pl^3}{48EI + 16kl^3}$$

中央Cのたわみ

荷重 $P$ による $x = l/2$ のたわみから $R_{\mathrm{B}}$ による $x = l/2$ のたわみを引きます。

片持ち梁の自由端に力 $R_{\mathrm{B}}$ が作用するとき、$x = l/2$ のたわみは:

$$\delta_{R,C} = \frac{R_{\mathrm{B}} (l/2)^2}{6EI}\left(3l - \frac{l}{2}\right) = \frac{5R_{\mathrm{B}} l^3}{48EI}$$

荷重 $P$ が $x = l/2$ に作用するとき、$x = l/2$ のたわみは:

$$\delta_{P,C} = \frac{P(l/2)^3}{3EI} = \frac{Pl^3}{24EI}$$

よって中央Cのたわみ:

$$y_C = \delta_{P,C} - \delta_{R,C} = \frac{Pl^3}{24EI} - \frac{5R_{\mathrm{B}} l^3}{48EI}$$

解答まとめ

片持ち梁＋ばね支持・中央集中荷重の公式

$$R_{\mathrm{B}} = \frac{5Pkl^3}{48EI + 16kl^3}$$ $$R_{\mathrm{A}} = P - R_{\mathrm{B}}$$ $$J_{\mathrm{A}} = \frac{Pl}{2} - R_{\mathrm{B}} l$$ $$y_B = \frac{R_{\mathrm{B}}}{k}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

固定端Aから右向きに $x$ を取り、各区間で左側の釣り合いを考えます。

区間1: $0 \le x < l/2$

$$V(x) = R_{\mathrm{A}}$$

区間2: $l/2 < x \le l$

$$V(x) = R_{\mathrm{A}} - P = -R_{\mathrm{B}}$$

$x = l/2$ で荷重 $P$ により不連続（$R_{\mathrm{A}}$ から $-R_{\mathrm{B}}$ へ $P$ だけ減少）。

曲げモーメント図（BMD）

各区間の曲げモーメント（梁上面引張を正）:

区間1: $0 \le x \le l/2$

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}} \cdot x$$

区間2: $l/2 \le x \le l$

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}} \cdot x + P\left(x - \frac{l}{2}\right)$$
  • $x = 0$: $M = J_{\mathrm{A}} = Pl/2 - R_{\mathrm{B}} l$
  • $x = l/2$: $M = J_{\mathrm{A}} - R_{\mathrm{A}} \cdot l/2$
  • $x = l$: $M = J_{\mathrm{A}} - R_{\mathrm{A}} \cdot l + Pl/2 = 0$（ばね支持端はモーメント自由）

ポイント

  • ばね支持を取り除いて静定な片持ち梁にするのが、重ね合わせ法の基本的な考え方。余剰拘束を1つ外して基本系を作り、適合条件で未知力を決定する。
  • $k \to \infty$ のとき: $R_{\mathrm{B}} \to \frac{5P}{16}$ となり、ばねが剛体（ローラー支持）の場合と一致する。これは一端固定・他端支持梁（propped cantilever）の解に一致する。
  • $k \to 0$ のとき: $R_{\mathrm{B}} \to 0$ となり、ばねがない場合（純粋な片持ち梁）に一致する。
  • 適合条件 $\delta_P - \delta_R = R_{\mathrm{B}} / k$ は「Bでのたわみ（下向き）= ばねの縮み」を意味している。物理的な意味を常に意識しよう。
  • ばね定数 $k$ と梁の剛性 $EI/l^3$ の比が、反力の配分を決める。両者の相対的な大きさが力学的挙動を支配する。
