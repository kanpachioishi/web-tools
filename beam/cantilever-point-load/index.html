片持ち梁の集中荷重（自由端）

固定端Aと自由端Bを持つ片持ち梁の自由端に集中荷重 $P$ が作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

目次

  1. 問題
  2. 解答
  3. 補足：せん断力図・曲げモーメント図
  4. ポイント

問題

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bを自由端とします。自由端Bに下向きの集中荷重 $P$ が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aから右方向に $x$ 軸を取るとき、以下を求めてください。

  1. 反力 $R_{\mathrm{A}}$ と固定端モーメント（反偶力）$J_{\mathrm{A}}$
  2. 自由端のたわみ $\delta_{\max}$

解答

反力の算出

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が発生します。力の釣り合いとモーメントの釣り合いから求めます。

  • 鉛直方向の力の釣り合い $$R_{\mathrm{A}} - P = 0 \quad \Rightarrow \quad R_{\mathrm{A}} = P$$
  • A点まわりのモーメントの釣り合い

    反時計回りを正とします。

    $$J_{\mathrm{A}} - P \cdot l = 0 \quad \Rightarrow \quad J_{\mathrm{A}} = Pl$$

たわみとたわみ角

たわみの微分方程式 $-EI\,y'' = M(x)$ を解きます。境界条件は固定端（$x = 0$）でたわみ $y = 0$、たわみ角 $y' = 0$ です。

曲げモーメント

位置 $x$ で梁を切断し、左側の釣り合いから:

$$M(x) = R_{\mathrm{A}}\,x - J_{\mathrm{A}} = Px - Pl = -P(l - x)$$

微分方程式の解

$$-EI\,y'' = -P(l - x)$$

1回積分してたわみ角:

$$-EI\,y' = -P\left(lx - \frac{x^2}{2}\right) + C_1$$

境界条件 $y'(0) = 0$ より $C_1 = 0$。

$$-EI\,y' = -P\left(lx - \frac{x^2}{2}\right)$$

もう1回積分してたわみ:

$$-EI\,y = -P\left(\frac{lx^2}{2} - \frac{x^3}{6}\right) + C_2$$

境界条件 $y(0) = 0$ より $C_2 = 0$。

結果

たわみ:

$$y(x) = \frac{Px^2}{6EI}(3l - x)$$

たわみ角:

$$\theta(x) = y'(x) = \frac{Px}{2EI}(2l - x)$$

最大たわみ（自由端 $x = l$）:

$$\delta_{\max} = y(l) = \frac{Pl^3}{3EI}$$

最大たわみ角（自由端 $x = l$）:

$$\theta_{\max} = \theta(l) = \frac{Pl^2}{2EI}$$

解答まとめ

片持ち梁・自由端集中荷重の解答

$$R_{\mathrm{A}} = P, \qquad J_{\mathrm{A}} = Pl$$ $$\delta_{\max} = \frac{Pl^3}{3EI}, \qquad \theta_{\max} = \frac{Pl^2}{2EI}$$

補足：せん断力図・曲げモーメント図

せん断力図（SFD）

任意の位置 $x$（$0 \le x \le l$）で梁を切断し、左側の釣り合いを考えます。

$$V(x) = R_{\mathrm{A}} = P \quad (\text{一定})$$

せん断力は梁全長にわたって $P$ で一定です。

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントは:

$$M(x) = R_{\mathrm{A}}\,x - J_{\mathrm{A}} = Px - Pl = -P(l - x)$$
  • $x = 0$（固定端A）: $M = -Pl$（ホギング最大）
  • $x = l$（自由端B）: $M = 0$

曲げモーメントは梁全長にわたって負（ホギング）で、固定端で絶対値最大 $Pl$ となり、自由端に向かって線形に $0$ へ近づきます。

ポイント

  • 片持ち梁の集中荷重は、材料力学で最も基本的な問題のひとつ。まずこの形を確実に解けるようにしましょう。
  • 固定端には鉛直反力と反偶力の2つの反力が発生します。
  • せん断力は一定、曲げモーメントは線形 — これは集中荷重の特徴です。
  • たわみの微分方程式では、境界条件の適用が積分定数を決めるカギになります。
  • $\delta_{\max} = \frac{Pl^3}{3EI}$ は暗記すべき基本公式です。
