SNSに画像や動画を投稿するとき、「画像が切れてしまう」「余白ができてしまう」と困ったことはありませんか？その原因のほとんどはアスペクト比の不一致です。この記事では、アスペクト比の基礎知識から各SNSの推奨サイズまで、まとめて解説します。

アスペクト比とは

アスペクト比とは、画像や動画の横と縦の長さの比率のことです。「16:9」「4:3」「1:1」のように、横:縦の比率で表現します。

たとえば「16:9」は、横の長さが16に対して縦の長さが9という意味です。具体的なピクセル数は異なっても、比率が同じであれば同じアスペクト比になります。

  • 1920 x 1080px → 16:9
  • 1280 x 720px → 16:9
  • 3840 x 2160px → 16:9

アスペクト比を正しく理解していれば、画像の切り抜きや余白の問題を防げます。

YouTube の推奨サイズ

YouTubeの動画やサムネイルには、以下のサイズが推奨されています。

種類推奨サイズアスペクト比
動画（フルHD）1920 x 1080px16:9
動画（4K）3840 x 2160px16:9
サムネイル1280 x 720px16:9
チャンネルアート2560 x 1440px16:9
ショート動画1080 x 1920px9:16

YouTubeのサムネイルは最低でも1280x720px以上、ファイルサイズは2MB以下が推奨です。カスタムサムネイルはクリック率に直結するため、16:9のアスペクト比を必ず守りましょう。

Instagram の推奨サイズ

Instagramは投稿の種類によって最適なアスペクト比が異なります。

種類推奨サイズアスペクト比
フィード（正方形）1080 x 1080px1:1
フィード（横長）1080 x 566px1.91:1
フィード（縦長）1080 x 1350px4:5
ストーリーズ1080 x 1920px9:16
リール1080 x 1920px9:16
プロフィール画像320 x 320px1:1

Instagramのフィード投稿で最もエンゲージメントが高いのは4:5の縦長フォーマットです。フィード上で表示面積が大きくなるため、スクロールを止めてもらいやすくなります。

X（Twitter）の推奨サイズ

X（旧Twitter）の画像投稿には以下のサイズが推奨されています。

種類推奨サイズアスペクト比
画像投稿（1枚）1200 x 675px16:9
画像投稿（2枚）700 x 800px（各）7:8
ヘッダー画像1500 x 500px3:1
プロフィール画像400 x 400px1:1
OGP画像1200 x 630px1.91:1

Xのタイムラインでは16:9の画像が最もきれいに表示されます。画像が複数枚の場合はトリミングされるため、重要な要素は中央に配置しましょう。

TikTok の推奨サイズ

TikTokは縦型動画がメインのプラットフォームです。

種類推奨サイズアスペクト比
動画1080 x 1920px9:16
プロフィール画像200 x 200px1:1

TikTokでは必ず9:16の縦型フルスクリーンで制作しましょう。画面上下にはUIが重なるため、重要なテキストや顔は中央の安全エリア（上下約150pxを避ける）に配置するのがポイントです。

リサイズのコツ

画像をリサイズする際に覚えておきたいポイントを紹介します。

アスペクト比を維持してリサイズする

アスペクト比を変えずにリサイズすれば、画像が歪むことはありません。たとえば1920x1080の画像を960x540にリサイズしても、16:9の比率は維持されます。

拡大よりも縮小を優先する

画像を拡大するとぼやけてしまいます。素材は常に必要なサイズより大きいものを用意し、縮小して使いましょう。

クロップ（切り抜き）のコツ

アスペクト比を変更する場合は、被写体の位置に注意してクロップしましょう。人物写真の場合は顔が切れないよう、中央〜やや上寄りに配置するのが定番です。

アスペクト比の計算が必要なときは、計算ツールを使うと横幅から高さを自動算出でき、面倒な計算が不要になります。

まとめ

アスペクト比は画像・動画制作の基本です。各SNSの推奨サイズを把握しておけば、画像が切れたり余白ができたりするトラブルを防げます。特にYouTubeは16:9、Instagramフィードは4:5、TikTokは9:16と覚えておきましょう。

リサイズの際は必ずアスペクト比を維持し、必要に応じてアスペクト比計算ツールを活用して正確なサイズを算出してください。
