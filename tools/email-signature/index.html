メール署名ジェネレーター

おしゃれなHTML署名を無料で簡単作成

名前・会社名・SNSリンクを入力するだけで、プロフェッショナルなメール署名を自動生成。5種類のテンプレートからデザインを選び、アクセントカラーもカスタマイズ可能。生成した署名はワンクリックでコピーして、GmailやOutlookにそのまま設定できます。

Gmail / Outlookへの設定方法

Gmail
Outlook
Thunderbird
  1. 上の「署名をコピー（Gmail用）」ボタンをクリックして署名をコピーします。
  2. Gmailを開き、右上の歯車アイコン → 「すべての設定を表示」をクリック。
  3. 「全般」タブをスクロールし、「署名」セクションを見つけます。
  4. 「新規作成」をクリックし、署名の名前を入力。
  5. 署名のテキストエリアにCtrl+V（Mac: Cmd+V）で貼り付けます。
  6. ページ下部の「変更を保存」をクリックして完了です。

※ HTMLソースではなく、「署名をコピー（Gmail用）」ボタンを使ってリッチテキストとして貼り付けてください。

  1. 上の「署名をコピー（Gmail用）」ボタンをクリックして署名をコピーします。
  2. Outlook Webの場合：右上の歯車アイコン → 「Outlookのすべての設定を表示」。
  3. 「メール」 → 「作成と返信」を選択。
  4. 「メールの署名」セクションで「+ 新しい署名」をクリック。
  5. 署名エディタにCtrl+V（Mac: Cmd+V）で貼り付けます。
  6. 「保存」をクリックし、デフォルトの署名として設定して完了です。

※ デスクトップ版Outlookの場合は「ファイル」→「オプション」→「メール」→「署名」から設定できます。

  2. Thunderbirdを開き、「ツール」 → 「アカウント設定」を選択。
  3. 対象のメールアカウントを選び、「HTML形式でメッセージを編集する」にチェック。
  4. 署名テキスト欄の下にある「HTML形式で記述する」にチェック。
