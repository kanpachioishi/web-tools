オームの法則とは？

オームの法則は、電気回路における電圧（V）、電流（I）、抵抗（R）の関係を表す基本法則です。1827年にドイツの物理学者ゲオルク・ジモン・オームが発見しました。この法則は電子工学・電気工学のあらゆる分野で使われ、回路設計や故障解析の基礎となります。

本ツールでは、3つの値のうち2つを入力するだけで残りの値と電力（W）を自動的に計算します。単位の自動変換や回路図表示にも対応しています。

オームの法則計算ツールの使い方

  1. 計算モードを選択 — 求めたい値に応じて「V と R から計算」「V と I から計算」「I と R から計算」のいずれかを選びます。
  2. 既知の値を入力 — 電圧・電流・抵抗の値を入力します。mV、kV、mA、μA、kΩ、MΩなどの単位も選択可能です。
  3. 計算結果を確認 — 「計算する」ボタンを押すか、入力と同時にリアルタイムで結果が表示されます。電力（W）も同時に算出されます。
  4. 回路図で確認 — 計算結果が回路図とオームの法則ホイールにも反映され、直感的に理解できます。

よくある質問

オームの法則はどんな場面で使いますか？

オームの法則は、LEDに接続する抵抗値の計算、電源回路の設計、配線の電圧降下の計算、ヒューズの選定など、電気・電子回路のあらゆる場面で使われます。電気工事士試験や電験（電気主任技術者試験）にも頻出する重要な法則です。

直列回路と並列回路での計算方法の違いは？

直列回路では抵抗値は合算（R合計 = R1 + R2 + ...）し、電流はどの部分でも同じです。並列回路では抵抗の逆数の合計の逆数（1/R合計 = 1/R1 + 1/R2 + ...）となり、各抵抗にかかる電圧は同じです。いずれの場合もオームの法則が適用されます。

単位の接頭辞（k、m、μ）はどう変換しますか？

kΩ（キロオーム）は1,000Ω、MΩ（メガオーム）は1,000,000Ωです。mA（ミリアンペア）は0.001A、μA（マイクロアンペア）は0.000001Aです。本ツールではプルダウンで単位を選べば自動変換されます。

電力の計算はなぜ重要ですか？

電力は抵抗やデバイスが消費するエネルギーを表し、部品の定格を超えると過熱・焼損の原因になります。例えば1/4W抵抗に0.5Wを流すと破損するため、電力計算は安全な回路設計に不可欠です。
