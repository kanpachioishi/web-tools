ボード線図とは？

ボード線図（Bode plot）は、制御工学や電子回路設計で広く使われる周波数応答の表現方法です。1938年にベル研究所のヘンドリック・ボードが考案しました。横軸に角周波数を対数スケール、縦軸にゲイン（dB）と位相（度）を取った2つのグラフから構成されます。

ボード線図を使うことで、システムの帯域幅、安定性余裕（ゲイン余裕・位相余裕）、共振特性などを視覚的に把握できます。フィルタ設計やフィードバック制御系の安定性解析に欠かせないツールです。

ボード線図ジェネレーターの使い方

  1. 伝達関数を入力 — 分子・分母の多項式係数を高次の項から順にカンマ区切りで入力します。プリセットから代表的な伝達関数を選ぶこともできます。
  2. 周波数範囲を設定 — 描画する周波数範囲（rad/s）を指定します。対数スケールで広い範囲をカバーできます。
  3. グラフを確認 — 上段にゲイン線図（dB）、下段に位相線図（度）が自動描画されます。
  4. 安定性指標を確認 — ゲイン余裕（GM）、位相余裕（PM）、交差周波数が自動計算されます。

ボード線図の読み方

ボード線図は上下2段のグラフで構成されます。上段がゲイン線図（振幅特性）、下段が位相線図（位相特性）です。横軸は対数スケールの周波数（rad/s）で、広い周波数帯域の特性を一度に確認できます。

安定性の判定

ゲイン余裕と位相余裕がともに正であれば、閉ループ系は安定です。一般的にゲイン余裕は6dB以上、位相余裕は30〜60度が推奨されます。

よくある質問

ゲイン余裕と位相余裕の違いは？

ゲイン余裕（GM）は位相が-180°になる周波数でのゲインの余裕（0dBまでの距離）を表します。位相余裕（PM）はゲインが0dBになる周波数での位相の余裕（-180°までの距離）を表します。どちらも正であればシステムは安定ですが、一般的にGM≧6dB、PM≧30°が推奨されます。

dB（デシベル）とは何ですか？

デシベル（dB）は対数スケールでゲインを表す単位です。ゲインGのdB表現は 20×log₁₀(G) です。例えば、ゲインが10倍なら20dB、100倍なら40dB、1/10なら-20dBとなります。対数スケールを使うことで、広い範囲のゲイン変化を一つのグラフで表現できます。

折点周波数とは何ですか？

折点周波数（コーナー周波数）は、ボード線図のゲイン特性が傾き変化する周波数です。1次遅れ系 1/(Ts+1) の場合、折点周波数は ω=1/T で、この周波数でゲインが約-3dB下がります。折点周波数を境にゲインの傾きが変化します。

伝達関数の係数はどのように入力しますか？

分子・分母の多項式の係数を最高次の項から順にカンマ区切りで入力します。例えば G(s) = 10/(s² + 3s + 2) の場合、分子に「10」、分母に「1, 3, 2」と入力します。プリセットから選択すると自動的に係数が入力されます。
