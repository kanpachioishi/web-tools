    cron式の書き方ガイド

    cron式の基本構造

    cron式は5つのフィールドで構成されています。各フィールドはスペースで区切ります。

    フィールド値の範囲説明
    分0-59実行する分
    時0-23実行する時（24時間制）
    日1-31実行する日
    月1-12実行する月
    曜日0-7実行する曜日（0,7=日曜）

    特殊文字

    • * - すべての値（例: 分に * → 毎分）
    • , - 複数の値を指定（例: 1,15 → 1日と15日）
    • - - 範囲を指定（例: 1-5 → 月曜〜金曜）
    • / - 間隔を指定（例: */5 → 5分ごと）

    よく使うcron式の例

    • * * * * * - 毎分実行
    • 0 * * * * - 毎時0分に実行
    • 0 0 * * * - 毎日午前0時に実行
    • 0 9 * * 1-5 - 平日の午前9時に実行
    • 0 0 1 * * - 毎月1日の午前0時に実行
    • */10 * * * * - 10分ごとに実行
