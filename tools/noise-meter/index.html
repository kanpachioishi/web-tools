騒音レベル測定

スマホ・PCで簡単デシベル測定

お使いのスマホやPCのマイクを使って、周囲の騒音レベル（デシベル）をリアルタイムに測定します。日常の騒音チェックや防音対策の参考にお使いください。無料・登録不要・ブラウザだけで完結します。

騒音レベルの目安

dB 環境の例 体感
20 dB 木の葉が触れ合う音、雪の降る音 ほぼ無音
30 dB 深夜の郊外、ささやき声 非常に静か
40 dB 図書館、静かな住宅地の昼間 静か
50 dB 静かなオフィス、エアコンの室外機 日常レベル
60 dB 通常の会話、テレビの音 普通
70 dB 掃除機、騒がしい街頭 やや騒がしい
80 dB 電車内、パチンコ店 騒がしい
90 dB カラオケ（店内）、犬の鳴き声 非常に騒がしい
100 dB 電車のガード下、ライブ会場 聴覚障害のリスク
110 dB 車のクラクション、工事現場 苦痛を感じる
120 dB ジェット機の離陸（近距離） 聴力損傷の危険
