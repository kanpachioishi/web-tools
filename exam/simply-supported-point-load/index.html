単純支持梁の集中荷重（中央）

左端Aをピン支持、右端Bをローラー支持とする単純支持梁の中央に集中荷重 $P$ が作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

  1. 問題設定

問題設定

長さ $l$ の単純支持梁において、左端Aをピン支持、右端Bをローラー支持とし、梁の中央（$x = l/2$）に下向きの集中荷重 $P$ が作用しています。梁の曲げ剛性は $EI$（一様）とします。

左端Aから右方向に $x$ 軸を取ります。

反力の算出

ピン支持Aとローラー支持Bにはそれぞれ鉛直反力 $R_{\mathrm{A}}$、$R_{\mathrm{B}}$ が発生します。力の釣り合いとモーメントの釣り合いから求めます。

  • B点まわりのモーメントの釣り合い

    反時計回りを正とします。

    $$R_{\mathrm{A}} \cdot l - P \cdot \frac{l}{2} = 0 \quad \Rightarrow \quad R_{\mathrm{A}} = \frac{P}{2}$$
  • 鉛直方向の力の釣り合い $$R_{\mathrm{A}} + R_{\mathrm{B}} - P = 0 \quad \Rightarrow \quad R_{\mathrm{B}} = P - \frac{P}{2} = \frac{P}{2}$$

対称性からも明らかなように、$R_{\mathrm{A}} = R_{\mathrm{B}} = P/2$ となります。

せん断力図（SFD）

任意の位置 $x$ で梁を切断し、左側の釣り合いを考えます。

左半分（$0 \le x < l/2$）:

$$V(x) = R_{\mathrm{A}} = \frac{P}{2}$$

右半分（$l/2 < x \le l$）:

$$V(x) = R_{\mathrm{A}} - P = \frac{P}{2} - P = -\frac{P}{2}$$

荷重作用点 $x = l/2$ でせん断力が $P/2$ から $-P/2$ に不連続に変化します。

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントは:

左半分（$0 \le x \le l/2$）:

$$M(x) = R_{\mathrm{A}} \cdot x = \frac{Px}{2}$$

右半分（$l/2 \le x \le l$）:

$$M(x) = R_{\mathrm{A}} \cdot x - P\left(x - \frac{l}{2}\right) = \frac{P(l-x)}{2}$$
  • $x = 0$, $x = l$: $M = 0$
  • $x = l/2$（中央）: $M_{\max} = \dfrac{Pl}{4}$

曲げモーメントは中央で最大値 $Pl/4$ となる三角形分布です。

たわみとたわみ角

たわみの微分方程式 $EI\,y'' = M(x)$ を解きます。対称性を利用して左半分（$0 \le x \le l/2$）のみ考えます。境界条件は $y(0) = 0$、$y'(l/2) = 0$（対称条件）です。

微分方程式の解

左半分（$0 \le x \le l/2$）:

$$EI\,y'' = \frac{Px}{2}$$

1回積分してたわみ角:

$$EI\,y' = \frac{Px^2}{4} + C_1$$

対称条件 $y'(l/2) = 0$ より:

$$0 = \frac{P(l/2)^2}{4} + C_1 \quad \Rightarrow \quad C_1 = -\frac{Pl^2}{16}$$ $$EI\,y' = \frac{Px^2}{4} - \frac{Pl^2}{16}$$

もう1回積分してたわみ:

$$EI\,y = \frac{Px^3}{12} - \frac{Pl^2 x}{16} + C_2$$

境界条件 $y(0) = 0$ より $C_2 = 0$。

結果

たわみ（$0 \le x \le l/2$、対称）:

$$y(x) = \frac{Px}{48EI}(3l^2 - 4x^2)$$

最大たわみ（中央 $x = l/2$）:

$$\delta_{\max} = y\!\left(\frac{l}{2}\right) = \frac{Pl^3}{48EI}$$

端部のたわみ角（$x = 0$）:

$$\theta_A = \theta_B = \frac{Pl^2}{16EI}$$

公式まとめ

単純支持梁・中央集中荷重の公式

$$R_{\mathrm{A}} = R_{\mathrm{B}} = \frac{P}{2}$$ $$V(x) = \begin{cases} \dfrac{P}{2} & (0 \le x < l/2) \\[6pt] -\dfrac{P}{2} & (l/2 < x \le l) \end{cases}$$ $$M(x) = \begin{cases} \dfrac{Px}{2} & (0 \le x \le l/2) \\[6pt] \dfrac{P(l-x)}{2} & (l/2 \le x \le l) \end{cases}$$ $$M_{\max} = \frac{Pl}{4}$$ $$y(x) = \frac{Px}{48EI}(3l^2 - 4x^2) \quad (0 \le x \le l/2)$$ $$\delta_{\max} = \frac{Pl^3}{48EI}$$

ポイント

  • 単純支持梁の中央集中荷重は、材料力学で最も基本的な問題のひとつです。片持ち梁と並んで、まず確実に解けるようにしましょう。
  • 対称性を利用すると、反力は自明に $P/2$ ずつとなり、たわみの境界条件も簡潔になります。
  • せん断力は荷重作用点で不連続に変化します。SFDが「階段状」になるのは集中荷重の特徴です。
  • 曲げモーメントは荷重作用点で最大となる三角形分布です。$M_{\max} = Pl/4$ は頻出公式です。
  • $\delta_{\max} = \dfrac{Pl^3}{48EI}$ は暗記すべき基本公式です。片持ち梁（$Pl^3/3EI$）と比較すると、単純支持の方がたわみがかなり小さいことがわかります。
