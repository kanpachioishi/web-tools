単純支持梁の等分布荷重

左端Aをピン支持、右端Bをローラー支持とする単純支持梁の全長にわたって等分布荷重 $q$ が作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

問題設定

長さ $l$ の単純支持梁において、左端Aをピン支持、右端Bをローラー支持とし、梁の全長にわたって下向きの等分布荷重 $q$（単位長さあたりの荷重）が作用しています。梁の曲げ剛性は $EI$（一様）とします。

左端Aから右方向に $x$ 軸を取ります。

反力の算出

ピン支持Aとローラー支持Bにはそれぞれ鉛直反力 $R_{\mathrm{A}}$、$R_{\mathrm{B}}$ が発生します。力の釣り合いとモーメントの釣り合いから求めます。

  • B点まわりのモーメントの釣り合い

    反時計回りを正とします。等分布荷重の合力 $ql$ は梁の中央 $x = l/2$ に作用します。

    $$R_{\mathrm{A}} \cdot l - ql \cdot \frac{l}{2} = 0 \quad \Rightarrow \quad R_{\mathrm{A}} = \frac{ql}{2}$$
  • 鉛直方向の力の釣り合い $$R_{\mathrm{A}} + R_{\mathrm{B}} - ql = 0 \quad \Rightarrow \quad R_{\mathrm{B}} = ql - \frac{ql}{2} = \frac{ql}{2}$$

対称性からも明らかなように、$R_{\mathrm{A}} = R_{\mathrm{B}} = ql/2$ となります。

せん断力図（SFD）

任意の位置 $x$（$0 \le x \le l$）で梁を切断し、左側の釣り合いを考えます。位置 $x$ までに作用する分布荷重は $qx$ なので:

$$V(x) = R_{\mathrm{A}} - qx = \frac{ql}{2} - qx = q\!\left(\frac{l}{2} - x\right)$$
  • $x = 0$（A端）: $V = ql/2$（最大正値）
  • $x = l/2$（中央）: $V = 0$
  • $x = l$（B端）: $V = -ql/2$（最大負値）

せん断力はA端で $ql/2$ から始まり、線形に減少して中央でゼロを通過し、B端で $-ql/2$ となります。

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントは:

$$M(x) = R_{\mathrm{A}} \cdot x - \frac{qx^2}{2} = \frac{qlx}{2} - \frac{qx^2}{2} = \frac{qx(l-x)}{2}$$
  • $x = 0$, $x = l$: $M = 0$
  • $x = l/2$（中央）: $M_{\max} = \dfrac{ql^2}{8}$

曲げモーメントは中央で最大値 $ql^2/8$ となる放物線分布です。

たわみとたわみ角

たわみの微分方程式 $EI\,y'' = M(x)$ を解きます。境界条件は $y(0) = 0$、$y(l) = 0$ です。

微分方程式の解

$$EI\,y'' = \frac{qx(l-x)}{2} = \frac{q}{2}(lx - x^2)$$

1回積分してたわみ角:

$$EI\,y' = \frac{q}{2}\left(\frac{lx^2}{2} - \frac{x^3}{3}\right) + C_1$$

もう1回積分してたわみ:

$$EI\,y = \frac{q}{2}\left(\frac{lx^3}{6} - \frac{x^4}{12}\right) + C_1 x + C_2$$

境界条件 $y(0) = 0$ より $C_2 = 0$。

境界条件 $y(l) = 0$ より:

$$0 = \frac{q}{2}\left(\frac{l^4}{6} - \frac{l^4}{12}\right) + C_1 l = \frac{ql^4}{24} + C_1 l$$ $$C_1 = -\frac{ql^3}{24}$$

結果

たわみ:

$$y(x) = \frac{qx}{24EI}(l^3 - 2lx^2 + x^3)$$

たわみ角:

$$\theta(x) = y'(x) = \frac{q}{24EI}(l^3 - 6lx^2 + 4x^3)$$

最大たわみ（中央 $x = l/2$）:

$$\delta_{\max} = y\!\left(\frac{l}{2}\right) = \frac{5ql^4}{384EI}$$

端部のたわみ角（$x = 0$）:

$$\theta_A = \theta_B = \frac{ql^3}{24EI}$$

公式まとめ

単純支持梁・等分布荷重の公式

$$R_{\mathrm{A}} = R_{\mathrm{B}} = \frac{ql}{2}$$ $$V(x) = q\!\left(\frac{l}{2} - x\right)$$ $$M(x) = \frac{qx(l-x)}{2}$$ $$M_{\max} = \frac{ql^2}{8}$$ $$y(x) = \frac{qx}{24EI}(l^3 - 2lx^2 + x^3)$$ $$\delta_{\max} = \frac{5ql^4}{384EI}$$

ポイント

  • 等分布荷重では、せん断力が線形、曲げモーメントが放物線（2次関数）になります。荷重を1回積分するとSFD、もう1回積分するとBMDになるという関係を意識しましょう。
  • $M_{\max} = ql^2/8$ は構造力学で最も頻出の公式のひとつです。確実に覚えましょう。
  • 最大たわみの公式 $\delta_{\max} = \dfrac{5ql^4}{384EI}$ の係数「$5/384$」は覚えにくいですが、非常に重要な公式です。
  • 等分布荷重 $q$ の合力は $P = ql$ です。中央集中荷重と同じ全荷重でも、分布荷重の方がたわみは小さくなります（$5ql^4/384EI$ vs $Pl^3/48EI = ql^4/48EI$）。
  • 対称性から $R_{\mathrm{A}} = R_{\mathrm{B}}$、$\theta_A = -\theta_B$ となり、最大たわみは中央に生じます。
