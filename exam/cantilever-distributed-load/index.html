片持ち梁の等分布荷重

固定端Aと自由端Bを持つ片持ち梁の全長にわたって等分布荷重 $q$ が下向きに作用する場合の反力・せん断力・曲げモーメント・たわみを求めます。

目次

  1. 問題設定
  2. 反力の算出
  3. せん断力図（SFD）
  4. 曲げモーメント図（BMD）

  6. 公式まとめ

問題設定

長さ $l$ の片持ち梁において、左端Aを固定端、右端Bを自由端とし、梁の全長にわたって下向きの等分布荷重 $q$（単位長さあたりの荷重）が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aから右方向に $x$ 軸を取ります。

反力の算出

固定端Aには鉛直反力 $R_{\mathrm{A}}$ と反偶力 $J_{\mathrm{A}}$ が発生します。力の釣り合いとモーメントの釣り合いから求めます。

  • 鉛直方向の力の釣り合い $$R_{\mathrm{A}} - ql = 0 \quad \Rightarrow \quad R_{\mathrm{A}} = ql$$
  • A点まわりのモーメントの釣り合い

    反時計回りを正とします。等分布荷重の合力 $ql$ は梁の中央 $x = l/2$ に作用します。

    $$J_{\mathrm{A}} - ql \cdot \frac{l}{2} = 0 \quad \Rightarrow \quad J_{\mathrm{A}} = \frac{ql^2}{2}$$

せん断力図（SFD）

任意の位置 $x$（$0 \le x \le l$）で梁を切断し、左側の釣り合いを考えます。位置 $x$ までに作用する分布荷重は $qx$ なので:

$$V(x) = R_{\mathrm{A}} - qx = ql - qx = q(l - x)$$
  • $x = 0$（固定端A）: $V = ql$（最大）
  • $x = l$（自由端B）: $V = 0$

せん断力は固定端で最大 $ql$ となり、自由端に向かって線形に減少してゼロになります。

曲げモーメント図（BMD）

任意の位置 $x$ での曲げモーメントは:

$$M(x) = J_{\mathrm{A}} - R_{\mathrm{A}} \cdot x + \frac{qx^2}{2} = \frac{ql^2}{2} - qlx + \frac{qx^2}{2} = \frac{q(l-x)^2}{2}$$
  • $x = 0$（固定端A）: $M = \frac{ql^2}{2}$（最大）
  • $x = l$（自由端B）: $M = 0$

曲げモーメントは固定端で最大 $\frac{ql^2}{2}$ となり、放物線を描いて自由端でゼロになります。

たわみとたわみ角

たわみの微分方程式 $EI\,y'' = M(x)$ を解きます。境界条件は固定端（$x = 0$）でたわみ $y = 0$、たわみ角 $y' = 0$ です。

微分方程式の解

$$EI\,y'' = \frac{q(l-x)^2}{2}$$

1回積分してたわみ角:

$$EI\,y' = -\frac{q(l-x)^3}{6} + C_1$$

境界条件 $y'(0) = 0$ より:

$$0 = -\frac{ql^3}{6} + C_1 \quad \Rightarrow \quad C_1 = \frac{ql^3}{6}$$ $$EI\,y' = \frac{ql^3}{6} - \frac{q(l-x)^3}{6} = \frac{q}{6}\left[l^3 - (l-x)^3\right]$$

もう1回積分してたわみ:

$$EI\,y = \frac{q}{6}\left[l^3 x + \frac{(l-x)^4}{4}\right] + C_2$$

境界条件 $y(0) = 0$ より:

$$0 = \frac{q}{6} \cdot \frac{l^4}{4} + C_2 \quad \Rightarrow \quad C_2 = -\frac{ql^4}{24}$$

結果

たわみ:

$$y(x) = \frac{qx^2}{24EI}(6l^2 - 4lx + x^2)$$

たわみ角:

$$\theta(x) = y'(x) = \frac{qx}{6EI}(3l^2 - 3lx + x^2)$$

最大たわみ（自由端 $x = l$）:

$$\delta_{\max} = y(l) = \frac{ql^4}{8EI}$$

最大たわみ角（自由端 $x = l$）:

$$\theta_{\max} = \theta(l) = \frac{ql^3}{6EI}$$

公式まとめ

片持ち梁・等分布荷重の公式

$$R_{\mathrm{A}} = ql, \qquad J_{\mathrm{A}} = \frac{ql^2}{2}$$ $$V(x) = q(l - x)$$ $$M(x) = \frac{q(l-x)^2}{2}$$ $$y(x) = \frac{qx^2}{24EI}(6l^2 - 4lx + x^2)$$ $$\delta_{\max} = \frac{ql^4}{8EI}, \qquad \theta_{\max} = \frac{ql^3}{6EI}$$

ポイント

  • 等分布荷重では、せん断力が線形、曲げモーメントが放物線（2次関数）になります。荷重を1回積分するとSFD、もう1回積分するとBMDになるという関係を意識しましょう。
  • 反力の計算では、等分布荷重の合力 $ql$ が梁の中央に作用すると考えてモーメントの釣り合いを立てます。
  • たわみの微分方程式を解く際、$(l-x)$ の形を含む積分に注意が必要です。符号の間違いが起きやすいポイントです。
  • $\delta_{\max} = \frac{ql^4}{8EI}$ は頻出公式です。集中荷重の $\frac{Pl^3}{3EI}$ と合わせて覚えておきましょう。
  • 等分布荷重 $q$ の合力は $P = ql$ です。同じ全荷重でも、集中荷重と等分布荷重ではたわみが異なる点に注意してください。
