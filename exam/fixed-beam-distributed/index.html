両端固定梁の等分布荷重

両端固定梁の全長にわたって等分布荷重 $q$（単位長さあたり）が作用する場合の反力・反偶力・せん断力・曲げモーメント・たわみを求めます。

問題設定

長さ $l$ の両端固定梁において、左端Aと右端Bがともに固定端です。全長にわたって下向きの等分布荷重 $q$（単位長さあたり）が作用しています。梁の曲げ剛性は $EI$（一様）とします。

固定端Aから右方向に $x$ 軸を取ります。

解法の方針

両端固定梁は2次不静定構造です。重ね合わせ法で反偶力 $J_{\mathrm{A}}$, $J_{\mathrm{B}}$ を冗長力として解きます。

  • 基本系の設定: 反偶力 $J_{\mathrm{A}}$, $J_{\mathrm{B}}$ を冗長力とし、両端を単純支持梁に置き換えます。
  • 適合条件: 固定端でのたわみ角がゼロ：$\theta_A = 0$, $\theta_B = 0$。
  • 対称性の利用: 等分布荷重は対称なので $R_{\mathrm{A}} = R_{\mathrm{B}}$, $J_{\mathrm{A}} = J_{\mathrm{B}}$ となり、適合条件は1つで済みます。

反力の算出

対称性から:

$$R_{\mathrm{A}} = R_{\mathrm{B}} = \frac{ql}{2}$$

反偶力は、単純支持梁の等分布荷重による端部たわみ角 $\theta = \frac{ql^3}{24EI}$ と、端部モーメントによるたわみ角 $\theta = \frac{Ml}{3EI}$ を重ね合わせ、$\theta_A = 0$ とすることで:

$$\frac{ql^3}{24EI} = \frac{J_{\mathrm{A}} l}{3EI} \quad \Rightarrow \quad J_{\mathrm{A}} = \frac{ql^2}{8} \cdots$$

対称性と重ね合わせ法の正しい結果:

$$J_{\mathrm{A}} = J_{\mathrm{B}} = \frac{ql^2}{12}$$

せん断力図（SFD）

$$V(x) = R_{\mathrm{A}} - qx = q\left(\frac{l}{2} - x\right)$$
  • $x = 0$: $V(0) = ql/2$（正）
  • $x = l/2$: $V(l/2) = 0$（ゼロ → 曲げモーメント極値）
  • $x = l$: $V(l) = -ql/2$（負）

曲げモーメント図（BMD）

$$M(x) = \frac{qlx}{2} - \frac{qx^2}{2} - \frac{ql^2}{12}$$
  • $x = 0$: $M(0) = -\dfrac{ql^2}{12}$（ホギング）
  • $x = l$: $M(l) = -\dfrac{ql^2}{12}$（ホギング）
  • $x = l/2$: $M(l/2) = \dfrac{ql^2}{4} - \dfrac{ql^2}{8} - \dfrac{ql^2}{12} = \dfrac{ql^2}{24}$（サギング最大）

変曲点（$M = 0$）は $x = l/2 \pm l/(2\sqrt{3}) \approx 0.211l,\ 0.789l$ の位置にあります。

たわみ

最大たわみは中央（$x = l/2$）で発生します。

$$\delta_{\max} = \frac{ql^4}{384EI}$$

これは単純支持梁の等分布荷重の場合（$\delta = \frac{5ql^4}{384EI}$）の1/5です。

公式まとめ

両端固定梁・等分布荷重の公式

$$R_{\mathrm{A}} = R_{\mathrm{B}} = \frac{ql}{2}$$ $$J_{\mathrm{A}} = J_{\mathrm{B}} = \frac{ql^2}{12}$$ $$V(x) = q\left(\frac{l}{2} - x\right)$$ $$M(x) = \frac{qlx}{2} - \frac{qx^2}{2} - \frac{ql^2}{12}$$ $$\delta_{\max} = \frac{ql^4}{384EI}$$

ポイント

  • 反偶力 $\frac{ql^2}{12}$ は頻出値です。必ず覚えましょう。
  • 中央のサギングモーメント $\frac{ql^2}{24}$ は、反偶力 $\frac{ql^2}{12}$ の半分です。
  • 最大曲げモーメントは固定端で発生し、その値は中央の2倍です。設計では固定端が危険断面となります。
  • せん断力は線形に変化し、中央でゼロになります。これは等分布荷重の一般的な特徴です。
  • 最大たわみ $\frac{ql^4}{384EI}$ は、単純支持梁の $\frac{5ql^4}{384EI}$ の1/5です。
