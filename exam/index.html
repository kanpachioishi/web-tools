梁の公式集

材料力学に登場する梁の基本パターンを構造別に整理しました。各ページで反力・せん断力図（SFD）・曲げモーメント図（BMD）・たわみの導出と公式を解説しています。

片持ち梁 片持ち梁 + ばね 固定・ローラー 単純支持梁 両端固定梁

片持ち梁

集中荷重（自由端） 自由端に集中荷重 P が作用 集中荷重（中央） 中央に集中荷重 P が作用 偏心集中荷重 任意の位置 al に集中荷重 P が作用 等分布荷重（全体） 全長にわたって等分布荷重 q が作用 等分布荷重（左半分） 固定端側の左半分に等分布荷重 q が作用 等分布荷重（右半分） 自由端側の右半分に等分布荷重 q が作用 三角形分布荷重（左端 q → 右端 0） 固定端側 q から自由端側 0 へ線形に減少 三角形分布荷重（左端 0 → 右端 q） 固定端側 0 から自由端側 q へ線形に増加 モーメント荷重（自由端） 自由端にモーメント M₀ が作用 モーメント荷重（中央） 中央にモーメント M₀ が作用

片持ち梁 + ばね支持

集中荷重（中央） ばね支持端 + 中央に集中荷重 P 等分布荷重 ばね支持端 + 全長に等分布荷重 q

一端固定・他端ローラー支持梁

集中荷重（中央） 中央に集中荷重 P が作用 等分布荷重 全長に等分布荷重 q が作用 モーメント荷重（中央） 中央にモーメント M₀ が作用 モーメント荷重（右端） ローラー端にモーメント M₀ が作用 三角形分布荷重（固定端 q → ローラー端 0） 固定端側 q からローラー端側 0 へ線形に減少 三角形分布荷重（固定端 0 → ローラー端 q） 固定端側 0 からローラー端側 q へ線形に増加

単純支持梁

集中荷重（中央） 中央に集中荷重 P が作用 偏心集中荷重 任意の位置 al に集中荷重 P が作用 等分布荷重 全長に等分布荷重 q が作用 モーメント荷重（中央） 中央にモーメント M₀ が作用

両端固定梁

集中荷重（中央） 中央に集中荷重 P が作用 偏心集中荷重 任意の位置 al に集中荷重 P が作用 等分布荷重（全体） 全長に等分布荷重 q が作用 等分布荷重（左半分） 左半分に等分布荷重 q が作用 山形分布荷重 中央 q・両端 0 の三角形分布荷重が作用
